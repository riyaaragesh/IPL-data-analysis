# IPL-data-analysis

## Overview
This project analyzes IPL match data using Python and data visualization libraries to uncover trends in team performance, toss decisions, venues, and player statistics.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Features
- Team win analysis
- Toss winner and toss decision analysis
- Venue analysis
- Season-wise match trends
- Top player of the match analysis
- Data visualization using graphs and charts

## Dataset
IPL Complete Dataset (2008–2020)

## Key Insights
- Identified teams with the highest number of wins.
- Analyzed the impact of toss decisions on match outcomes.
- Found the most frequently used IPL venues.
- Visualized season-wise IPL match trends.
- Identified players with the most Player of the Match awards.

## Sample Visualizations
- Bar Charts
- Pie Charts
- Trend Graphs

## Project Outcome
This project demonstrates exploratory data analysis, data cleaning, visualization, and storytelling using real-world sports datasets.
